# 🧑🏻‍💻 캠픽
#### 캠핑 중고차 거래 앱

# 🧾 기능
### 회원가입
##### - 일반 사용자와 딜러 중 사용자 타입 선택 필요
##### - 약관 동의 후 이메일 인증, 비밀번호 확인 필요
##### - 휴대전화 인증 필요
##### - 프로필 이미지, 닉네임 등록 필요
### 홈
##### - 광고 배너, 이벤트 조회 가능
##### - 빠른 매물 조회 가능
##### - 빠른 카테고리 별 매물 조회 가능
##### - 추천 매물(신규, 인기) 조회 가능
### 매물 등록
##### - 이미지, 매물명, 연식, 차량종류/모델, 판매지역, 주행거리, 차량번호, 판매가격, 차량옵션 상세 설명을 기입하여 매물 등록 가능 
### 매물 조회
##### - 사용자들이 등록한 매물 조회 가능
##### - 최근 등록순, 낮은 가격순, 높은 가격순, 주행거리 짧은 순, 최신 연식순으로 매물 필터링 가능
##### - 가격, 주행거리, 연식, 차량 종류로 매물 필터링 가능
### 찜목록 제공
##### - 내가 찜한 매물 조회 가능
### 채팅 기능
##### - 매물을 통해 채팅 전송 가능
##### - 채팅방 목록과 채팅방을 통해 상대방 온라인 여부 확인 가능
##### - 상대방과 실시간 채팅 가능
### 내 프로필
#### 등록한 매물 관리
##### - 판매중, 판매완료 매물 확인 가능
#### 회원정보 수정
##### - 닉네임, 프로필 이미지, 자기소개, 연락처 변경 가능
#### 회원 설정
##### - 비밀번호 변경
##### - 로그아웃
##### - 회원탈퇴 가능
